# popous_react
>藉由文章或照片的分享,讓大家的生活串連起來<br />
>  * 使用 node.js , react , react-router<br />
![Alt text](/public/img/popous.png)<br />

>## 功能
>  * 建立帳號<br />
>  * 變更個人資料<br />
>  * 發文(圖像/文字)<br />


>## 特色
>  * 變更頭像(自動以最小邊長裁切中央正方形後儲存)<br />
>  * masonry 文章排版(照發文時間排序)<br />
>  * RWD設計<br />
